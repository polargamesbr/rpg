Sobre os Efeitos
Esta página demonstra diferentes técnicas para criar efeitos visuais em HTML:
  • Canvas Particles: Sistemas de partículas desenhados em tempo real
  • CSS Animations: Animações performáticas usando apenas CSS
  • DOM Overlays: Elementos HTML posicionados sobre os cards
  • Hybrid: Combinação de todas as técnicas
Performance
Todos os efeitos rodam direto no navegador sem necessidade de assets externos. Canvas é ideal para efeitos dinâmicos, enquanto CSS é melhor para animações simples e repetitivas.
